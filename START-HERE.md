## üöÄ Quick Commands

### Setup
```bash
npm install                    # Install dependencies
npm run fetch-token           # Get authentication token
npm run verify:setup          # Verify installation
```

### Testing
```bash
npm test                      # Run all tests
npm run test:crud            # CRUD tests only
npm run test:security        # Security tests only
npm run test:performance     # Performance tests only
```

### Schema Management
```bash
npm run schema:production:ready    # Update everything
npm run swagger:advanced:fetch     # Fetch Swagger docs
npm run schema:harmonize:ids       # Harmonize IDs
```

### Validation
```bash
npm run schema:enhance:validate    # Validate schemas
npm run verify:refactoring         # Verify refactoring
```
